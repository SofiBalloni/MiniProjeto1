# Mini Projeto — Web Scraper de Livros

Este projeto coleta informações do site **Books to Scrape**, pegando:
título, preço, disponibilidade, nota e link do livro.

Os dados são salvos em **CSV** e **JSON** dentro da pasta `data`.

## Como usar

1️⃣ Clone o repositório:

```bash
git clone https://github.com/seu-usuario/nome-do-repo.git
cd nome-do-repo
```
